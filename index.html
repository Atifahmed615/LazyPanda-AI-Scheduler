💬 WhatsApp Reminders
Get class reminders directly on WhatsApp. This feature requires a small backend server to relay messages via the WhatsApp Business API.
⚠️ Backend Required — WhatsApp messages cannot be sent directly from a browser for security reasons. You need to set up a relay server.

Option A — Self-hosted (free):
1. Clone the Lazy Panda WhatsApp Server repo
2. Deploy to Railway / Render / Vercel (free tier)
3. Add your Twilio WhatsApp API key
4. Paste your server URL below

Option B — Use Twilio directly:
Sign up at twilio.com, get a WhatsApp-enabled number, and connect below.
